## 개요

현 레파지토리는 대용량 3D 데이터 시각화 엔진인 VIZCore3D.NET 제품(컨트롤)에 대한 기본 사용 예제 및 활용 코드를 공유하기 위한 목적입니다.

상세 API에 대한 설명 및 예제는 SHDC (http://www.softhills.net/SHDC) 를 참고 하시기 바랍니다.
